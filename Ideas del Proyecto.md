&nbsp;	4. Auxilium ( Mi version de los Jokers ).







Logica del Videojuego:



 	1.Empiezas el Sondeo.



 	2. Se genera un tablero con minas.



 	3. El jugador tiene 1 sola vida al inicio, pero puede comprar varias hasta llegar a 10.



 	4. Al descubrir las Casillas:

 		4.1. Casilla num = suma score ( dependiendo de que numero sea, si el num es 1 se le suma 2 al score y asi mismo hasta llegar al 8 ).

 		4.2. Casilla vacia = abre zona ( mientras mas dificil de vuelva menos zonas vacias Habra ).

 		4.3. Casilla mina = se termina el juego o se le quita una vida.

 

 	5. El score se multiplica con el multiplicador ( el score y el multiplicador se puede aumentar mas con los comodines, como los jokers).



 	6. Si el score es > o = al objetivo se pasa al siguiente Sondeo/Drill, la Cata/Spur o la siguiente Calicata/Pit.



 	7. Antes de que se pase al siguietne nivel, se ara una parada en la tienda donde estaran todos los comodides que pueden ayudarte a ganar mas facil.
